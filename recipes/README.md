# recipes/

Hand-seeded starter library lands here in Phase 1. Each recipe is a
single JSON file named `<id>.json`. Schema is defined in
[`../ARCHITECTURE.md`](../ARCHITECTURE.md) §Recipe format.

Phase 1 target: ~20 recipes spanning lofi, ambient, electronic, and
nature-hybrid (text-prompted) styles. Each ear-validated against a
real work session.

In Phase 2 this directory becomes the local cache of the canonical
Lockstep library repo on GitHub. Local edits to recipes here are not
synced upstream — to contribute a recipe, open a PR against that repo.
